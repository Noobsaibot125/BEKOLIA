# BEKOLIA – Site Web Professionnel

## Installation

### 1. Backend (Node.js)
```bash
cd Backend
npm install
npm run dev
```
Le serveur démarre sur http://localhost:5000

### 2. Frontend (React)
```bash
cd Frontend
npm install
npm run dev
```
Le site démarre sur http://localhost:5173

## Structure
- Backend/ → API Node.js + Express + SQLite
- Frontend/ → React + Vite + Tailwind CSS
- Frontend/Images/ → Vos logos et vidéos

## Pages
- / → Accueil
- /services → Nos Services
- /galerie → Galerie Photos/Vidéos
